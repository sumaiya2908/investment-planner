### Steps to Install:
1. Run the command to clone the repository
```sh
git clone https://github.com/sumaiya2908/investment-planner.git
```

2. Change the directory
```sh
cd invesment-planner
```

3. Install the dependencies and start the server
```sh
npm install
npm run dev
```
